You can run the scripts by typing :
Python "name of the file"
examples:
"pyhton flutter.py"
### functionalities
exec.py ----> create a folder with the desired project name and inside the folder will create a "read me" file 
flutter.py -----> create a folder with the desired project name + read me file + creating a flutter project (name of the flutter project is the same as the folder name )
dj.py ------> create a folder with the desired project name + read me file + creating a django project (the name of the django project is the same as the folder name)
